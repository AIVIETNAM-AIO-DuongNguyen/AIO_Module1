# Research notes index

This directory holds research notes that the project relies on: library API
details, method background, benchmark specifics, and design decisions worth
recording. This file is the **index / manager** for those notes.

## How to use this directory

- One note per topic, as a Markdown file in this directory.
- Every note is written in English and states, near the top, what it covers and
  what it is used for in this project.
- Whenever a note is added, changed, or removed, update the table below so the
  index always reflects the current set of notes.
